# 山手線3D互動式乘搭模擬器 (Yamanote Line 3D Interactive Simulator)

東京山手線的沉浸式3D列車乘搭模擬體驗。

## 核心功能

### 1. 列車模擬
- ✅ 逼真的列車加速、減速和速度曲線
- ✅ 真實的車站停靠時間間隔
- ✅ 完整的山手線29個車站名稱和順序
- ✅ 動態乘客密度模擬
- ✅ 列車門開關機制與音效反饋

### 2. 視覺體驗
- ✅ 第一人稱視角（車廂內部或外部）
- ✅ 動態車窗景色與沿線地標
- ✅ 逼真的光照效果（晝夜循環）
- ✅ 車站月台細節與標識
- ✅ 乘客擁擠程度視覺化
- ✅ 離開車廂進入月台行走

### 3. 音頻系統
- ✅ 車站播報系統（日語 + 英語選項）
- ✅ 真實列車音效
- ✅ 環境背景音
- ✅ 導航背景音樂

### 4. 互動元素
- ✅ 虛擬座位選擇
- ✅ 車門操控
- ✅ 路線信息顯示
- ✅ 實時時刻表追蹤
- ✅ 乘客觀察互動機制

### 5. 遊戲模式
- 🎮 遊客模式：教育體驗 + 車站信息
- 🚆 通勤者模式：逼真的日常通勤模擬
- 🗺️ 自由探索：任何車站上下車
- 🎯 挑戰模式：完成特定目標任務

## 技術棧

- **引擎**: Three.js / Babylon.js
- **前端**: React / Vue.js
- **狀態管理**: Redux / Vuex
- **音頻**: Web Audio API
- **物理引擎**: Cannon-es / Rapier
- **資料庫**: Firebase / MongoDB

## 專案結構

```
yamote-sim/
├── src/
│   ├── components/       # React/Vue 元件
│   ├── scenes/          # 3D 場景
│   ├── systems/         # 列車模擬系統
│   ├── audio/           # 音頻管理
│   ├── data/            # 車站資料、時刻表
│   └── utils/           # 工具函數
├── public/              # 靜態資源、模型、音效
├── tests/               # 測試檔案
└── package.json
```
